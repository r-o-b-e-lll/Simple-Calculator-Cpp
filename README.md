C++ Simple Calculator

A functional command-line interface (CLI) calculator developed in C++. This project demonstrates fundamental programming concepts including loops, switch-case logic, and basic mathematical operations.

✨ Features
- **Basic Arithmetic:** Addition, Subtraction, Multiplication, and Division.
- **Advanced Math:** Square root, nth root, and Power ($x^y$) calculations.
- **Error Handling:** Built-in checks to prevent division by zero and square roots of negative numbers.
- **Continuous Execution:** Uses a `do-while` loop to allow multiple calculations in one session.
